# 🚀 Scripts Directory

Quick reference for running the application:

## Linux/Mac (macOS)
```bash
./scripts/run-dev.sh           # Run both client & server
./scripts/run-client.sh        # Client only
./scripts/run-server.sh        # Server only
./scripts/run-client-network.sh # Client with network mode
./scripts/run-client-render.sh # Client for Render deployment
```

## Windows (CMD)
```cmd
scripts\run-dev.bat
scripts\run-client.bat
scripts\run-server.bat
scripts\run-client-network.bat
```

## Windows (PowerShell)
```ps1
.\scripts\run-dev.ps1
.\scripts\run-client.ps1
.\scripts\run-server.ps1
```
